## 1. Modo Indicativo

Se usa para expresar hechos reales, objetivos y concretos. Es el modo que usamos en nuestro día a día para informar.

| **Tiempo Simple (Una palabra)**           | **Ejemplo** |
| ----------------------------------------- | ----------- |
| **Presente**                              | canto       |
| **Pretérito imperfecto** (Copretérito)    | cantaba     |
| **Pretérito perfecto simple** (Pretérito) | canté       |
| **Futuro simple**                         | cantaré     |
| **Condicional simple**                    | cantaría    |

| **Tiempo Compuesto (Haber + Participio)** | **Ejemplo**    |
| ----------------------------------------- | -------------- |
| **Pretérito perfecto compuesto**          | he cantado     |
| **Pretérito pluscuamperfecto**            | había cantado  |
| **Pretérito anterior**                    | hube cantado   |
| **Futuro compuesto**                      | habré cantado  |
| **Condicional compuesto**                 | habría cantado |
## 2. Modo Subjuntivo

Se usa para expresar deseos, dudas, probabilidades, emociones o situaciones hipotéticas (cosas que no son necesariamente hechos concretos).

| **Tiempo Simple**        | **Ejemplo**         |
| ------------------------ | ------------------- |
| **Presente**             | cante               |
| **Pretérito imperfecto** | cantara _o_ cantase |
| **Futuro simple**        | cantare             |

| **Tiempo Compuesto**             | **Ejemplo**                 |
| -------------------------------- | --------------------------- |
| **Pretérito perfecto compuesto** | haya cantado                |
| **Pretérito pluscuamperfecto**   | hubiera _o_ hubiese cantado |
| **Futuro compuesto**             | hubiere cantado             |
## 3. Modo Imperativo

Se utiliza exclusivamente para dar órdenes, consejos, ruegos o instrucciones. Solo tiene un tiempo (presente) y no se conjuga en primera persona del singular (yo).

| **Tiempo Simple** | **Ejemplo** | **Tiempo**                 | **Ejemplo** |
| ----------------- | ----------- | -------------------------- | ----------- |
| **Presente**      | Afirmativo  | ** Su proprio**            |             |
| **Presente**      | Negativo    | **Presente de Subjuntivo** |             |


### Las formas no personales

Se llaman así porque no indican quién hace la acción (yo, tú, nosotros) ni cuándo ocurre (pasado, presente, futuro). Son la base para construir otras cosas:

- **Infinitivo:** Es el nombre del verbo (_cantar, comer, vivir_).
- **Gerundio:** Expresa una acción en proceso (_cantando, comiendo, viviendo_).
- **Participio:** Expresa una acción terminada y se usa para los tiempos compuestos que vimos antes (_cantado, comido, vivido_).

### Las perífrasis verbales (El "falso" tiempo)

En español, a veces unimos dos verbos para crear un significado nuevo. Si estudias inglés, esto te sonará al _Present Continuous_ o al _Near Future_, pero en español la gramática oficial no los cuenta como tiempos propios. Los más importantes son:

- **Estar + Gerundio (Acción en progreso):** En lugar de decir "yo como", muchas veces decimos **"estoy comiendo"**.
- **Ir a + Infinitivo (Futuro próximo):** En el habla cotidiana, casi nadie dice "yo comeré" (Futuro simple). Lo más común es decir **"voy a comer"**.
- **Tener que + Infinitivo (Obligación):** Como en **"tengo que estudiar"**.

Así que, con la lista de tiempos que ya tienes, más las formas no personales y un par de estas fórmulas (perífrasis), ya conoces toda la estructura del español.
